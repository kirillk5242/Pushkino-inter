# Pushkino Inter

Интерактивная карта города Пушкино с информацией о достопримечательностях, маршрутах и услугах.

## Описание проекта

Pushkino Inter - это веб-приложение, предоставляющее интерактивную карту города Пушкино с подробной информацией о:

- Достопримечательностях и интересных местах
- Маршрутах общественного транспорта
- Местных услугах и организациях
- Событиях и мероприятиях

## Технологии

- Frontend: React.js с TypeScript
- Карты: Leaflet / OpenStreetMap
- Стилизация: CSS Modules / Styled Components
- Сборка: Vite
- Контроль версий: Git
